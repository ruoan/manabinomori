# manabinomori

## 開発

```bash
npm run dev
```

## ビルド

```bash
npm run build
```

## GitHub Pages デプロイ

`main` ブランチにプッシュすると GitHub Actions が自動でビルド・デプロイする。

**初回設定（リポジトリ作成後に一度だけ必要）:**
1. GitHub リポジトリの Settings → Pages を開く
2. Source を `GitHub Actions` に設定して Save

デプロイ先: https://ruoan.github.io/manabinomori/
